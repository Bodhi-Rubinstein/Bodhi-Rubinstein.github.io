My Projects

HPC Clusters

I work on HPC clusters at the National Center for Atmospheric Research (NCAR). In addition to helping maintain our prouduction clusers, Derecho and Casper, I also have built a research and development cluster with 15 compute nodes that will be used as a test system for Casper. I used xCAT as a system manager, PBS as the schedular, and Ansible for configuration management. All clusters run Linux and my work on these projects involves mostly Linux administration on different distributions.

Chameleon Cloud

I administer and maintain NCAR's Chameleon Cloud resource: the Thunder cluster. Chameleon Cloud is an NSF project providing cloud hosted HPC resources to researchers. This project involves managing the cluster using Openstack inside Chameleon Cloud's CHI-in-a-box implementation, monitoring usage, tracking applicable CVE's to maintain security, and working with the dev team to coordinate upgrades, maintenance, alerts, and troubleshoot hardware.

Brainrender

I worked with the Brainrender tool to visualize neural data for Root Lab. I visualized data from the Ventral Tegmental Area of a variety of neuronal subtypes as well as visualizing their density estimates. In addition, I plotted the projections and inputs of those neurons from other areas of the brain.

CEBRA Model

I used the CEBRA tool to create a learning model of the various neuronal subtypes in the Ventral Tegmental Area. The model could have one subtype removed from the VTA and the learning model of certain tasks will shift accordingly. This generalized model of the VTA helps express the differences in function between subtypes
